# **App Name**: Apex InvoiceGen

## Core Features:

- Dynamic Invoice Data Input: Users can input and edit all variable invoice details including customer information, MPO details, invoice numbers, dates, and core company details (header/footer).
- Product Line Item Editor: An interactive table for adding, editing, and deleting individual product line items including Product ID, Description, Pack Size, Unit Trade Price (TP), VAT Rate, and Quantity.
- Automated Invoice Calculations: Real-time calculation and display of 'Total TP', 'Total VAT', and 'Total Price' for each line item, as well as the overall 'Total Amount' and 'Net Payable Amount'.
- AI-Assisted Invoice Drafting Tool: A generative AI tool that can create an initial draft of an invoice by interpreting a natural language request, populating customer details and basic product lines.
- Invoice Preview and Finalization: Display a live preview of the complete invoice in a structured layout identical to the provided template, ready for review and finalization.
- Print and PDF Export Functionality: Enable users to print the generated invoice directly or export it as a high-fidelity PDF document, maintaining the A4 size and layout for professional output.

## Style Guidelines:

- Primary Color: A vibrant, bold red (#E31E24) to maintain brand recognition and highlight key elements, reflecting the energetic nature of a robust enterprise.
- Background Color: An extremely light, desaturated off-white with a subtle red tint (#FAF8F8) provides a clean, professional canvas that supports readability, suitable for official documents.
- Accent Color: A rich, deeper magenta (#A32B66) offering a strong contrast to the primary red while maintaining an analogous harmony, used for subtle highlights and interactive elements.
- Body and Headline Font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, which is ideal for displaying both large titles and intricate tabular data effectively across various screen sizes and print media.
- Minimal and Functional Icons: Use simple, clean line icons for actions like 'add item', 'delete item', 'print', and 'export', ensuring they are clear and do not distract from the document's professional nature.
- Structured A4 Document Layout: Adhere strictly to an A4 page dimension (210mm x 297mm) with well-defined sections (header, information grids, dynamic table, totals, footer, warranty), optimized for print legibility and digital display.
- Subtle UI Feedback: Employ gentle transition animations for actions such as adding a new line item, updating calculations, or switching between views, enhancing user interaction without being distracting or frivolous.
